Database Administration Summary

I have hands on experience in designing and creating databases using Microsoft Access, SQL Server, and MySQL programs. The tasks I have done was creating databases and tables; store, manipulate, and delete data. Examples of tasks are data querying, data inserting, data searching, data sorting, data deleting and so on. One other place I have created databases is as a backend data storage for desktop programs.

Databases are extremely helpful when data needs to be oraginzed by the database program itself and the software that uses the database continues to extract and access the data. In this case the database is not only managed by the software, a database administrator can control the information stored in the database as well.
