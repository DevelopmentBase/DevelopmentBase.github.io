System Administration Summary

In my role as System Administrater, my main mission was to administer, configure, and manage computer systems, servers, and other security devices. My task in server administering was creating and managing users and their data by Active Directory Domain Services (ADDS) and ensuring the server hardware components are working properly. Monitoring computers through Server Management and Active Directory consoles from the server, was my another duty.

There was times I had to upgrade or replace differenct hardware parts of a server or a computer. For example, replacing the RAM and Disk Storage for upgrade purpose then I had to check their correct installation from system administrative consoles.

Another tool that I often use is PowerShell. A command line tool that I use to monitor running apps and services, and overall system performance. When I need to work efficient and see the result quicker, I use PowerShell.
